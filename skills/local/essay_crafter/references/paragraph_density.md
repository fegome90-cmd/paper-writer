# Paragraph Density Gate

Use this pass to prevent overloaded paragraphs.

## One-job rule

Each paragraph should perform one rhetorical job:

- introduce a claim
- analyze evidence
- explain a limitation
- compare positions
- transition to the next section
- synthesize implications

If a paragraph changes jobs midway, split it.

## Practical heuristic

A strong paragraph usually contains:

1. one controlling idea
2. one main support cluster
3. one closing or transition move

Anything beyond that is a rewrite candidate.

## Split triggers

Split the paragraph when:

- a new citation cluster introduces a different analytical function
- the paragraph pivots from “what the evidence shows” to “what institutions should do”
- the paragraph moves from diagnosis to prescription
- the paragraph contains more than one natural topic sentence

## Review questions

- What is this paragraph trying to do?
- Could I name that job in five words?
- Does the last sentence belong to the same job as the first?

If the answer to the last question is “no”, split it.
