# Paraphrasing Guide

Use this reference when restating source material in your own words.

## Proper paraphrasing technique

1. Read the original until you understand it fully.
2. Set the original aside. Write the idea from memory.
3. Compare your version against the original. If more than three consecutive words match (excluding articles/prepositions), rewrite.
4. Add a signal phrase attributing the idea to its source.
5. Cite the source.

## Patchwriting avoidance

Patchwriting is swapping synonyms while keeping the original sentence structure. It is not paraphrasing.

Red flags:

- same clause order as the original
- only synonyms changed, grammar untouched
- paragraph-level structure mirrors the source

Fix: restructure the sentence. Change voice (active/passive), split or merge clauses, reorder information.

## Synthesis vs plagiarism boundary

Synthesis weaves ideas from multiple sources into an original analytical frame. Plagiarism presents one source's ideas as your own.

Rules:

- One source, your words, cited = paraphrase (acceptable)
- One source, their words, cited = direct quote (acceptable)
- One source, their structure with swapped words = patchwriting (not acceptable)
- Multiple sources, your analytical frame, all cited = synthesis (acceptable)

## Signal phrases for attribution

- X argues that ...
- According to X, ...
- X's analysis suggests ...
- As X demonstrates, ...
- X contends / maintains / observes that ...
- Building on X's finding that ..., this essay ...

## Decision tree: Is this paraphrase or plagiarism?

1. Did you write it from memory without the source visible? -> YES: likely paraphrase. Verify and cite. -> NO: go to 2.
2. Did you keep the original sentence structure? -> YES: patchwriting risk. Restructure completely. -> NO: go to 3.
3. Are three or more consecutive words identical to the source? -> YES: rewrite those phrases. -> NO: go to 4.
4. Did you add a signal phrase and citation? -> YES: acceptable paraphrase. -> NO: add attribution now.

When in doubt, treat it as a direct quote: use quotation marks and cite.
