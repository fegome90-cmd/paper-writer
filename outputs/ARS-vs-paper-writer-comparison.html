2. Arquitectura: Diferencia Fundamental

ARS: Prompt-Driven Architecture

ARS es un skill pack: cada “agente” es un prompt Markdown detallado que se ejecuta dentro de Claude Code. Los scripts Python son herramientas de verificación post-hoc, no el motor de escritura.

ARS = [42 prompts de agentes] + [81 scripts de verificación] + [20+ YAML schemas]
      ↑ LLM ejecuta los prompts           ↑ Python valida outputs

paper-writer: Engine-Driven Architecture

Nosotros somos una aplicación Python con motor propio de validación, reglas declarativas, clientes API reales, y un harness de orchestration. Los skills son un layer encima del motor.

paper-writer = [Motor Python] → [Validadores] → [Reglas YAML] → [CLI] → [Skills]
               ↑ Código real        ↑ 20 validators   ↑ 20 rule files

Conclusión clave: Son complementarios, no competitivos. ARS le dice al LLM cómo hacer la investigación. paper-writer verifica programáticamente que el output sea correcto.

5. Conclusión Estratégica

ARS y paper-writer son complementarios, no competitivos.

  • ARS es un skill pack de prompts para Claude Code: le dice al LLM cómo hacer investigación académica, con agentes especializados y un pipeline de 10 stages. Su fortaleza es la cobertura del workflow académico (research → write → review → revise → finalize).

  • paper-writer es un motor de validación programático: verifica que el output académico sea correcto usando reglas declarativas, clientes API reales, y validadores concretos. Su fortaleza es la corrección verificable sin depender del LLM.

El camino natural es:

  1. Portar los gaps de ARS como módulos Python en nuestro motor (arXiv client, temporal verification, PRISMA, contamination signals)
  2. Mantener nuestra ventaja en validación programática (ARS no tiene nada equivalente a nuestro motor de reglas)
  3. Opcionalmente integrar los agentes de ARS como skills importados, usando nuestro motor como backend de verificación

El resultado sería un sistema que combina la amplitud del workflow de ARS con la profundidad de validación de paper-writer.
